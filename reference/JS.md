# Mark character strings as literal JavaScript code

See [`htmlwidgets::JS()`](https://rdrr.io/pkg/htmlwidgets/man/JS.html)
for details.

## Usage

``` r
JS(...)
```

## Arguments

- ...:

  character vectors as the JavaScript source code (all arguments will be
  pasted into one character string)

## Value

A string that will be interpreted as JavaScript code in htmlwidgets.
