# World CO2 emissions

This dataset contains world polygons with CO2 emissions.

## Usage

``` r
co2_world
```

## Format

A data frame with 495 observations and 11 variables:

- `iso_code` : ISO code A3 for country.

- `name` : Name of country.

- `co2` : Annual CO2 emissions - Annual total emissions of carbon
  dioxide (CO2), excluding land-use change, measured in million tonnes.

- `co2_per_capita` : Annual CO2 emissions (per capita) - Annual total
  emissions of carbon dioxide (CO2), excluding land-use change, measured
  in tonnes per person.

- `geometry` : Geographical attributes.
