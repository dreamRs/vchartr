# Countries GDP

These data represent the GDP of the world's countries, classified by
continent and sub-region. This is a subset of the dataset
`rnaturalearth::countries110`.

## Usage

``` r
countries_gdp
```

## Format

A data frame with 177 observations and 3 variables:

- `REGION_UN` : Continent

- `SUBREGION` : Sub-region in the continent

- `ADMIN` : Administrative name of country

- `GDP_MD` : GDP

## Source
