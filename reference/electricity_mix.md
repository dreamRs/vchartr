# Electricity mix for 10 countries

This dataset represents the electricity mix of 10 countries (those with
the highest electricity generation) in 2023.

## Usage

``` r
electricity_mix
```

## Format

A data frame with 70 observations and 3 variables:

- `country` : Country name

- `source` : source of electricity

- `generation` : Total electricity generation - Measured in
  terawatt-hours.

- `type` : Low carbon or fossil fuels type of source.
