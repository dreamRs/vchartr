# Top CRAN downloads

The dataset contains data about CRAN downloads retrieved with
[cranlogs::cran_top_downloads](https://r-hub.github.io/cranlogs/).

## Usage

``` r
top_cran_downloads
```

## Format

A data frame with 100 observations and 5 variables.

## Source

[cranlogs](https://r-hub.github.io/cranlogs/)
